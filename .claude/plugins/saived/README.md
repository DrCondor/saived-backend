# sdlc plugin

SAIVED SDLC workflow skills.

## Skills

- `saived:tdd` — write failing test first
- `saived:pr` — open GH PR with diff-aware checklist
- `saived:parallel` — fan out to N implementer subagents in worktrees
- `saived:verification` — run rails test + rubocop + brakeman
- `saived:retro` — log session insights to `docs/superpowers/retros/`
- `saived:debug` — systematic debugging with Rails recipes

## Authoring conventions

Skills live in `skills/<name>/SKILL.md`. Frontmatter requires `name` and `description`.
